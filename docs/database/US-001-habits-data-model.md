# US-001 - Habits Data Model

## Table: habits

| Column        | Type      | Constraints                                   | Purpose                          |
|---------------|-----------|-----------------------------------------------|----------------------------------|
| `id`          | `BIGINT`  | PRIMARY KEY, GENERATED BY DEFAULT AS IDENTITY | однозначное определение привычки |
| `name`        | `TEXT`    | NOT NULL, UNIQUE INDEX uq_habits_name_lower   | название привычки                |
| `description` | `TEXT`    | может быть `NULL`                             | описание привычки                |
| `active`      | `BOOLEAN` | не `NULL`, значение по-умолчанию - `TRUE`     | активна ли привычка              |

## Database Constraints

- уникальный индекс `uq_habits_name_lower` по выражению `lower(name)` обеспечивает уникальность названия без учета регистра

## Application Rules

- перед сохранением приложение удаляет пробельные символы в начале и конце `name`
- пустое или состоящее только из пробельных символов имя не сохраняется
