# US-003 - List Habits

## Related User Story

[US-003 - Получение списка привычек](../user-stories/US-003-list-habits.md)

## Endpoint

- **Method:** `GET`
- **Path:** `/api/v1/habits`

## Successful Response

### Status

`200 OK`

### Body

Есть хотя бы одна активная привычка

```json
[
  {
    "id": 1,
    "name": "string",
    "description": "string",
    "completedToday": false
  }
]
```

Нет ни одной активной привычки

```json
[]
```

### Response Fields

| Field            | Type             | Description                                          |
|------------------|------------------|------------------------------------------------------|
| `id`             | `integer`        | ID привычки                                          |
| `name`           | `string`         | Название привычки                                    |
| `description`    | `string \| null` | Описание привычки или null, если описание не указано |
| `completedToday` | `boolean`        | Выполнена ли привычка за текущий день                |

## Business Rules

- Привычка либо активна, либо архивирована
- Для каждого дня хранится отдельный факт выполнения привычки
- Привычка считается выполненной с момента выполнения до конца текущего дня
- Часовой пояс используется Europe/Moscow
- API возвращает только активные привычки
- Архивированные привычки в список не включаются
- completedToday имеет значение true, если для привычки существует факт выполнения за текущую дату в Europe/Moscow; иначе false.
- Порядок элементов списка не гарантируется

## Out of Scope

- Пагинация, фильтрация, сортировка
