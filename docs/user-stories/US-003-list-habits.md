# US-003 - Просмотр списка привычек

## User Story
**Как** пользователь,  
**я хочу** просматривать список всех активных привычек,  
**чтобы** видеть все привычки, которые нужно выполнять.

## Acceptance Criteria

### Сценарий 1 - Просмотр списка активных привычек

**Given** существует хотя бы одна активная привычка  
**And** существует хотя бы одна архивированная привычка  
**When** пользователь запрашивает список активных привычек  
**Then** пользователь получает список активных привычек  
**And** список не содержит архивированных привычек  
**And** каждый элемент списка содержит название и информацию о выполнении за текущий день  
**And** каждый элемент списка содержит описание, если оно было указано

### Сценарий 2 - Нет активных привычек

**Given** в системе нет активных привычек  
**When** пользователь запрашивает список активных привычек  
**Then** пользователь получает пустой список
