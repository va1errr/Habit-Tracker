# US-005 - Архивирование привычки

## User Story
**Как** пользователь,  
**я хочу** архивировать привычку,  
**чтобы** убрать из списка привычку, которую больше не выполняю.

## Acceptance Criteria

### Сценарий 1 - Архивирование существующей активной привычки

**Given** привычка существует и активна  
**When** пользователь архивирует привычку  
**Then** привычка становится архивированной  
**And** привычка больше не отображается в списке активных привычек

### Сценарий 2 - Привычка не существует

**Given** привычка с указанным ID не существует  
**When** пользователь пытается архивировать привычку  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 3 - Привычка существует и архивирована

**Given** привычка существует и архивирована  
**When** пользователь пытается архивировать привычку  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 4 — Архивирование привычки с сохранёнными выполнениями

**Given** активная привычка имеет сохранённые факты выполнения  
**When** пользователь архивирует привычку  
**Then** привычка становится архивированной  
**And** сохранённые факты выполнения не удаляются
