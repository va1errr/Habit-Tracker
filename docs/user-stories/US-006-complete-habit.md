# US-006 - Выполнение привычки

## User Story
**Как** пользователь,  
**я хочу** отметить привычку выполненной за текущий день,  
**чтобы** зафиксировать свой прогресс.

## Acceptance Criteria

### Сценарий 1 - Выполнение существующей активной невыполненной привычки

**Given** привычка существует, активна и не выполнена за текущий день  
**When** пользователь отмечает привычку выполненной  
**Then** создаётся факт выполнения привычки за текущий день  
**And** привычка отображается выполненной за текущий день

### Сценарий 2 - Привычка не существует

**Given** привычка с указанным ID не существует  
**When** пользователь пытается выполнить привычку  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 3 - Привычка существует и архивирована

**Given** привычка существует и архивирована  
**When** пользователь пытается выполнить привычку  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 4 - Привычка существует, активна и выполнена

**Given** привычка существует, активна и выполнена за текущий день  
**When** пользователь пытается выполнить привычку  
**Then** пользователь получает сообщение о невозможности выполнить уже выполненную привычку
