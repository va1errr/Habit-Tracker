# US-004 - Изменение привычки

## User Story
**Как** пользователь,  
**я хочу** изменить привычку,  
**чтобы** поддерживать актуальное название и описание.

## Acceptance Criteria

### Сценарий 1 — Изменение существующей активной привычки с названием и описанием

**Given** привычка существует и активна  
**And** другой привычки с новым названием не существует  
**When** пользователь указывает привычке новое название и описание  
**Then** привычка изменяется  
**And** сохраняются переданные название и описание

### Сценарий 2 — Привычка не существует

**Given** привычка с указанным ID не существует  
**When** пользователь пытается изменить привычку  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 3 — Изменение существующей активной привычки без описания

**Given** привычка существует, активна и имеет описание  
**And** другой привычки с новым названием не существует  
**When** пользователь изменяет привычку, указывая только название  
**Then** привычка изменяется  
**And** привычка не имеет описания

### Сценарий 4 — Название не указано

**Given** привычка существует и активна  
**And** пользователь не указал название  
**When** пользователь пытается изменить привычку  
**Then** привычка не изменяется  
**And** пользователь получает сообщение об обязательном названии

### Сценарий 5 — Название состоит из пробельных символов

**Given** привычка существует и активна  
**And** пользователь указал название, состоящее только из пробельных символов  
**When** пользователь пытается изменить привычку  
**Then** привычка не изменяется  
**And** пользователь получает сообщение об обязательном названии

### Сценарий 6 — Пробелы по краям названия

**Given** привычка существует и активна  
**And** другой привычки с новым названием не существует  
**When** пользователь указывает привычке новое название с пробельными символами в начале или конце названия  
**Then** привычка изменяется  
**And** пробельные символы в начале и конце сохранённого названия отсутствуют

### Сценарий 7 — Повторяющееся название

**Given** привычка существует и активна  
**And** существует другая активная или архивированная привычка с таким названием без учёта регистра  
**When** пользователь указывает название, принадлежащее другой привычке  
**Then** привычка не изменяется  
**And** пользователь получает сообщение о повторяющемся названии

### Сценарий 8 — Привычка архивирована

**Given** привычка существует и архивирована  
**When** пользователь пытается изменить привычку  
**Then** привычка не изменяется  
**And** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 9 — Название не меняется

**Given** активная привычка существует  
**When** пользователь передаёт прежнее название и новое описание  
**Then** привычка изменяется  
**And** новое описание сохраняется
