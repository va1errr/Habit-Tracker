# US-007 - Отмена выполнения привычки

## User Story
**Как** пользователь,  
**я хочу** убрать отметку о выполнении привычки за текущий день,  
**чтобы** исправить ошибочно отмеченное выполнение.

## Acceptance Criteria

### Сценарий 1 - Отмена выполнения существующей активной выполненной привычки

**Given** привычка существует, активна и выполнена за текущий день  
**When** пользователь отменяет выполнение привычки  
**Then** факт выполнения привычки за текущий день удаляется  
**And** привычка отображается невыполненной за текущий день

### Сценарий 2 - Привычка не существует

**Given** привычка с указанным ID не существует  
**When** пользователь пытается отменить выполнение привычки  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 3 - Привычка существует и архивирована

**Given** привычка существует и архивирована  
**When** пользователь пытается отменить выполнение привычки  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 4 - Привычка существует, активна и не выполнена

**Given** привычка существует, активна и не выполнена за текущий день  
**When** пользователь пытается отменить выполнение привычки  
**Then** пользователь получает сообщение о невозможности отменить выполнение ещё не выполненной привычки
