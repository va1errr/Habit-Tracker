# US-002 - Просмотр привычки

## User Story
**Как** пользователь,  
**я хочу** просмотреть привычку,  
**чтобы** увидеть подробную информацию о ней.

## Acceptance Criteria

### Сценарий 1 - Просмотр существующей активной привычки

**Given** привычка существует и активна  
**When** пользователь запрашивает информацию о привычке  
**Then** пользователь получает название привычки  
**And** пользователь получает описание, если оно было указано  
**And** пользователь получает информацию о выполнении привычки за текущий день

### Сценарий 2 - Привычка не существует

**Given** привычка с указанным ID не существует  
**When** пользователь запрашивает информацию о привычке  
**Then** пользователь получает сообщение об отсутствии запрашиваемой привычки

### Сценарий 3 - Привычка существует и архивирована

**Given** привычка существует и архивирована  
**When** пользователь запрашивает информацию о привычке  
**Then** пользователь не получает данные привычки  
**And** пользователь получает сообщение об отсутствии запрашиваемой привычки
