# Habit Tracker

## Target User

Обычный человек, желающий следить за своими привычками.

## Problem

Неудобно вести учет своих привычек, отмечать выполнение и следить за прогрессом и статистикой.

## Solution

Создание API приложения, позволяющего вести учет своих привычек просто.

## MVP v1

- Создание привычки
- Просмотр привычки
- Просмотр списка привычек
- Изменение привычки
- Удаление привычки
- Выполнение привычки
- Отмена выполнения привычки за текущий день
- Один пользователь
- Все привычки ежедневные

## MVP v2

- Просмотр истории
- Просмотр статистики
- Категории привычек
- Аутентификация
- Восстановление удаленной привычки

## Later

- Управление длительностью привычки
- Календарь
- Достижения
- Экспорт данных

## Out of Scope

- Frontend
- Интеграция ИИ

## Business Rules

- Название привычки обязательно
- Названия не чувствительны к регистру. При получении названия API удаляет все пробельные символы в начале и конце строки
- Описание привычки необязательно
- Нельзя создать две привычки с одинаковым названием (даже если привычка архивирована)
- Привычка либо активна, либо архивирована
- Для каждого дня хранится отдельный факт выполнения привычки
- Выполнение можно отметить только один раз за день
- Привычка считается выполненной с момента выполнения до конца текущего дня
- Часовой пояс используется Europe/Moscow
- Удаление привычки означает архивирование
- Архивированная привычка не видна в общем списке, ее нельзя выполнить
- API отвечает 404 на получение архивированной привычки
- Архивированную привычку нельзя изменить
- У архивированную привычки нельзя отметить или отменить выполнение
- Архивирование привычки не удаляет сохранённые факты выполнения
