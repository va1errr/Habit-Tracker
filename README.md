# Habit Tracker

Backend API для создания ежедневных привычек, отметки их выполнения и отслеживания прогресса.

## MVP v1

- Создание привычки
- Просмотр привычки
- Просмотр списка привычек
- Изменение привычки
- Удаление привычки
- Выполнение привычки
- Отмена выполнения привычки за текущий день

Первая версия рассчитана на одного пользователя. Все привычки являются ежедневными.
